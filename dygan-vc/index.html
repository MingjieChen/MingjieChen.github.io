Improving Efficiency of GAN Voice Conversion using VQ-Wav2vec Features and Dynamic Convolution

Authors: Mingjie Chen, Yanghao Zhou, Heyan Huang, Thomas, Hain

Abstract: This paper presents a novel efficient voice conversion (VC) model based on generative adversarial network. Most of the state of the art (SOTA) VC models requires plenty of parameters, which limits applications of VC systems in practical situations. This paper proposes DYGAN-VC, aiming at improving model efficiency without huge performance degradation. To improve efficiency, DYGAN-VC uses pretrained VQWav2vec features as inputs. Besides, DyGAN-VC uses dynamic convolution, which enhances speech content modeling. DYGAN-VC reaches comparable performance of the SOTA with much less parameters. DYGAN-VC has only 7 million parameters and training finishes within 1 hour on one gpu.

Dataset

This paper uses the VCC2020 track 1 dataset.

Models

  • Source, Target: Natural speech of the source and target speakers.
  • Cascade ASR-TTS : a Transformer-based ASR + a Transformer-based TTS
  • Proposed: The proposed method with vq-wav2vec for M2O VC only using data from the target speaker.
